== Description ==

Xell is a plugin that allows you to connect your WordPress with the services of https://xell.shop/. By installing Xell, you grant permission to the services of xell.shop to access data from your store, such as: Products, Customers, Orders, Shipments, Discounts, Users, etc.

Xell es un plugin que permite contectar tu wordpress con los servicios de https://xell.shop/
Al installar Xell le das permisos a los servicios de xell.shop para haceder a datos de tu tienda, como:
Productos, Clientes, Ordenes, Envios, Descuentos, Usuarios, etc.

== Frequently Asked Questions ==

= What is Xell for? =
Xell is a plugin that uses the "Connect WooCommerce with your API" plugin to link your site with the Xell application.
https://xell.shop/

= Para que sirve Xell? =
Xell es un plugin que usa el plugin "Connect Woocommerce with your api" para conectar tu Sitio con la aplicacion Xell.
https://xell.shop/
